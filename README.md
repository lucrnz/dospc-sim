# DosPC Sim

Terminal UI application built with Python and Textual.
This project exposes an SSH server to connect to a simulated DOS environment.

## Installation

This project uses `uv` for dependency management:

```bash
uv sync
```

## Usage

Run the application:

```bash
uv run dospc-sim
```

## Development

Running unit tests:

```bash
uv run pytest
```
